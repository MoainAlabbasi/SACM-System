# S-ACM - نظرة عامة على النظام

## مقدمة

**S-ACM** (Smart Academic Content Management) هو نظام متكامل لإدارة المحتوى الأكاديمي مصمم خصيصاً للجامعات والمؤسسات التعليمية. يوفر النظام منصة موحدة لإدارة المقررات الدراسية، الملفات التعليمية، والتواصل بين المدرسين والطلاب، مع ميزات ذكاء اصطناعي محلية متقدمة.

## الرؤية

> "تمكين المؤسسات التعليمية من إدارة المحتوى الأكاديمي بكفاءة عالية مع الاستفادة من تقنيات الذكاء الاصطناعي المحلية"

## الأهداف الرئيسية

1. **مركزية المحتوى**: توفير مكان واحد لجميع المواد التعليمية
2. **سهولة الوصول**: واجهة بسيطة وسهلة الاستخدام لجميع الأدوار
3. **الذكاء الاصطناعي**: تلخيص تلقائي وتوليد أسئلة بدون اعتماد على خدمات خارجية
4. **الأمان**: نظام صلاحيات متعدد المستويات
5. **التواصل**: إشعارات فورية بين المدرسين والطلاب

## المستخدمون المستهدفون

| الدور | الوصف |
|-------|-------|
| **مدير النظام (Admin)** | إدارة كاملة للنظام والمستخدمين |
| **المدرس (Instructor)** | رفع المحتوى وإدارة المقررات |
| **الطالب (Student)** | الوصول للمحتوى واستخدام ميزات AI |

## الميزات الرئيسية

### 1. إدارة المستخدمين
- تسجيل دخول آمن
- تفعيل الحساب عبر OTP
- إعادة تعيين كلمة المرور
- ثلاثة أدوار مختلفة

### 2. إدارة المحتوى الأكاديمي
- المقررات الدراسية
- الفصول الدراسية
- التخصصات والمستويات
- الملفات (PDF, DOCX, PPTX, فيديو, صوت)

### 3. الذكاء الاصطناعي المحلي
- تلخيص المحاضرات
- توليد أسئلة اختبارية
- مساعد ذكي (Chatbot)
- يعمل بدون إنترنت بعد التثبيت

### 4. نظام الإشعارات
- إشعارات داخلية
- إشعار تلقائي عند رفع ملف جديد
- إشعارات من المدرسين

```
│                      Django 5.x (Python)                     │
├──────────────┬──────────────┬──────────────┬────────────────┤
│   Accounts   │     Core     │  AI Service  │  Notifications │

├──────────────┴──────────────┴──────────────┴────────────────┤

└─────────────────────────────────────────────────────────────┘
```
